# 🏡 spiti-alert

Αυτόματες ειδοποιήσεις email για **νέες αγγελίες ενοικίασης σπιτιού** στα δυτικά / βορειοδυτικά της Θεσσαλονίκης, μέσω Spitogatos (Apify scraper `fayoussef/spitogatos-scraper`).

> ⚠️ **Γιατί είναι PUBLIC:** Το GitHub στα **private** repos (δωρεάν tier) παρατάει/ακυρώνει τα προγραμματισμένα (cron) τρεξίματα όταν έχει φόρτο — στην πράξη έτρεχε ~1 στα 5. Στα **public** repos τα τρέχει αξιόπιστα (απεριόριστα δωρεάν λεπτά). Γι' αυτό το κάναμε public. Κανένα προσωπικό δεδομένο δεν εκτίθεται: τα email των παραληπτών ζουν στο secret `RECIPIENTS`, όχι στον κώδικα.

## Πώς δουλεύει

1. **GitHub Actions** τρέχει 3 φορές τη μέρα (08:17, 13:17, 19:17 ώρα Ελλάδας) — ή χειροκίνητα από το tab Actions → Run workflow.
2. Καλεί τον Apify actor να σκανάρει 3 αναζητήσεις του Spitogatos (μόνο ενοικιάσεις, ≤ €900):
   - Θεσσαλονίκη — Υπόλοιπο Νομού (Κουφάλια, Άγ. Αθανάσιος, Νέα Μεσημβρία…)
   - Θεσσαλονίκη — Περιφερειακοί δήμοι (Ωραιόκαστρο, Πεύκα, Φίλυρο…)
   - Νομός Κιλκίς (για Νέο Αγιονέρι)
3. **Σκληρά φίλτρα** (κόβουν): τιμή ≤ €900, εμβαδόν ≥ 100 τ.μ., **παλαιότητα**: κατασκευή από το 2000 και μετά — ΕΚΤΟΣ αν είναι ανακαινισμένο από το 2015+ (άγνωστο έτος δεν κόβεται, περνάει σημαδεμένο «θέλει έλεγχο»), και **γεωγραφικά**: η αγγελία (βάσει συντεταγμένων) πρέπει να πέφτει μέσα σε «πράσινη ζώνη» και έξω από κάθε «κόκκινη ζώνη» (Σίνδος, ΒΙ.ΠΕ., ΣΙΔΕΝΟΡ, Αγχίαλος, Χαλκηδόνα, Άδενδρο). Οι ζώνες ορίζονται στο `config.json`.
4. **Βαθμολόγηση 0–100** (δεν κόβει — ζυγίζει): τετραγωνικά (στόχος 120+, μπόνους 150/180+), **υπνοδωμάτια (στόχος 3)**, μπάνια/WC, οικόπεδο/αυλή, γκαράζ, **ενεργειακή κλάση**, **θέρμανση** (μέσο+αυτονομία+ενδοδαπέδια), **φόρτιση EV**, τζάμια διπλά/τριπλά, πόσο κάτω από το ταβάνι είναι η τιμή, απόσταση από δουλειά & σχολείο, κήπος/τζάκι/AC/ηλιακός/αποθήκη/θέα/μπαλκόνι, τύπος (κατοικία), έτος κατασκευής/ανακαίνισης. Αν λείπει κάποια παροχή, η αγγελία **δεν απορρίπτεται** — χάνει πόντους και το email γράφει τι λείπει. Επιπλέον σημάνσεις χωρίς πόντους: 💬 συζητήσιμη τιμή, 💡 τιμή με λογαριασμούς, ⏳ καιρό στην αγορά (>60 μέρες). Σκληρά κόβονται όσα «χρήζουν ανακαίνισης» ή είναι ημιτελή. Στέλνονται όσες έχουν **σκορ ≥ 45** (`scoring.minScore`), ταξινομημένες: ⭐⭐⭐ ≥70 · ⭐⭐ 50–69 · ⭐ <50.

> **Σημείωση αποκωδικοποίησης:** η ενεργειακή κλάση (κωδικοί 1–10 → Α+…Η) είναι σειριακή αντιστοίχιση υψηλής βεβαιότητας. Το μέσο θέρμανσης (1=Πετρέλαιο, 2=Φυσικό αέριο, 13=Αντλία θερμότητας κ.λπ.) είναι **εμπειρική** αντιστοίχιση — οι άγνωστοι κωδικοί εμφανίζονται ως «κωδ. Χ». Αν δεις σε αγγελία λάθος ετικέτα, διορθώνεται στο `HEATING_MEDIUM` του `src/index.js`.
5. Κρατάει στο `state/seen.json` ποιες αγγελίες έχει ήδη στείλει — **email έρχεται ΜΟΝΟ για καινούργιες**.
6. Το email πάει σε bill@emmanuela.gr + emma@emmanuela.gr (μέσω Microsoft Graph), με σκορ, φωτογραφία, τιμή, τ.μ., οικόπεδο, παροχές, «τι λείπει», αποστάσεις από δουλειά/σχολείο, link αγγελίας και χάρτη.

## Δοκιμή βαθμολόγησης σε αποθηκευμένο δείγμα

```powershell
node src/score-test.js path\to\listings.json
```

## Secrets που χρειάζεται (Settings → Secrets and variables → Actions)

| Secret | Τι είναι |
|---|---|
| `APIFY_TOKEN` | Token του Apify λογαριασμού |
| `RECIPIENTS` | Email παραληπτών, comma-separated (π.χ. `bill@…,emma@…`) — εκτός κώδικα γιατί public |
| `SMTP_HOST` | π.χ. `smtp.office365.com` ή `smtp.gmail.com` |
| `SMTP_PORT` | `587` (ή `465`) |
| `SMTP_USER` | Ο λογαριασμός αποστολής |
| `SMTP_PASS` | Κωδικός / app password |
| `MAIL_FROM` | (προαιρετικό) διεύθυνση αποστολέα, αλλιώς = SMTP_USER |

Χωρίς SMTP secrets τρέχει σε **dry-run**: γράφει στο log τι θα έστελνε, δεν στέλνει τίποτα.

## Αλλαγή κριτηρίων

Όλα στο `config.json`: `maxPrice`, `minSqMeters`, ζώνες με ακτίνες σε χλμ, παραλήπτες, URLs αναζήτησης. **Προσοχή:** αν αλλάξει το `maxPrice`, άλλαξε το `maxprice-900` και μέσα στα `startUrls`.

## Τοπικό τρέξιμο (δοκιμή)

```powershell
$env:APIFY_TOKEN = "..."   # χωρίς SMTP secrets → dry-run
node src/index.js
```

## Κόστος

Apify: $1 / 1.000 αγγελίες. Κάθε τρέξιμο σκανάρει ~100–300 αγγελίες → **λίγα ευρώ τον μήνα** το πολύ (η δωρεάν μηνιαία πίστωση του Apify συνήθως το καλύπτει).
